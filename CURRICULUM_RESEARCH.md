## 🏫 主要塾のカリキュラム分析

### 公文式（KUMON）

#### 特徴
- **スモールステップ方式**
- 学年に関係なく、個人の理解度に応じて進む
- 計算力の徹底強化

#### カリキュラム構成
1. **足し算・引き算**: 徹底的な反復練習
2. **掛け算・割り算**: 九九の完全習得
3. **分数・小数**: 段階的な導入
4. **方程式**: 中学レベルの計算

#### マスマジへの応用
- 各ワールド内で難易度を細かく分ける
- 「基礎」「標準」「応用」の3段階設定
- 習熟度に応じた自動レベル調整

---

### 学研教室

#### 特徴
- **教科書準拠**
- 学校の進度に合わせた学習
- 文章題の強化

#### カリキュラム構成
1. 学校の進度に完全対応
2. 文章題の解法パターン学習
3. 図形問題の視覚的理解

#### マスマジへの応用
- ストーリー問題（文章題）を重視
- 図形問題には図解を追加
- 学年別に教科書の単元順に対応

### 栄光ゼミナール

#### 特徴
- **中学受験対応**
- 応用問題・思考力問題
- 集団授業形式

#### カリキュラム構成
1. 基礎→標準→応用の段階的学習

## 💡 学年判定システム

### プレイヤー作成時の学年選択
- 選択した学年に応じて、適切なワールドを表示
- 下の学年のワールドも選択可能（復習用）
- 上の学年のワールドはロック（チャレンジモードで解放可能）

### ワールド表示ロジック
```javascript
function getAvailableWorlds(playerGrade) {
    const allWorlds = WORLD_DATABASE;
    
    return allWorlds.filter(world => {
        // 自分の学年以下のワールドを表示
        return world.targetGrade <= playerGrade;
    }).sort((a, b) => {
        // 学年順、難易度順にソート
        return a.targetGrade - b.targetGrade || a.difficulty - b.difficulty;
    });
}
```

## 🎮 ワールド設計の原則

### 1. 学習指導要領完全準拠
- 各学年の学習内容に完全対応
- 文科省の指導要領を100%カバー

### 2. 段階的な難易度設定
- 各ワールド内で「基礎」「標準」「応用」の3段階
- プレイヤーの正答率に応じて自動調整

### 3. ストーリー性の重視
- 各ワールドに独自のストーリー
- RPGとしての世界観を維持

### 4. 実用性の確保
- 実生活で使える問題
- 文章題を中心に構成

### 5. つまずき防止
- 苦手単元は複数のワールドで反復
- ヒントシステムで挫折させない
