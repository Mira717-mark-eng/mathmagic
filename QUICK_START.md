## 🔧 404エラーが出る場合の対処法

### 症状
```
404 (Not Found)
character-select.html が見つかりません
```

### 原因
1. **ブラウザの履歴**から古いページにアクセスしている
2. **ブックマーク**が古いURLを指している
3. **ブラウザのキャッシュ**が残っている

### 解決方法

#### ステップ1: ブックマーク・履歴を削除
ブックマークやブラウザ履歴から`character-select.html`を削除してください。

#### ステップ2: 正しいURLでアクセス
```
https://mira717-mark-eng.github.io/mathmagic/
```
↑このURLをブックマークしてください

#### ステップ3: キャッシュクリア
- **Windows**: `Ctrl` + `Shift` + `Delete`
- **Mac**: `Cmd` + `Shift` + `Delete`

または、キャッシュクリアツールを使用：
```
https://mira717-mark-eng.github.io/mathmagic/clear-cache.html
```
